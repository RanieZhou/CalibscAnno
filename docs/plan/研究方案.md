
## 1. 研究定位

拟定题目：

> **CalibscAnno: Foundation model-based open-set cell type annotation with calibrated rejection**

中文：

> **基于单细胞基础模型表征与校准拒识的开放集细胞类型注释方法**

核心目标不是做普通细胞类型注释，而是做：

> **开放集细胞类型注释 / 未知细胞类型识别**

也就是训练集中只有已知细胞类型，测试集中可能出现训练集没有见过的细胞类型。模型需要做到：

```text
1. 对已知细胞类型尽量准确注释；
2. 对未知细胞类型不要强行错标；
3. 能根据置信度或距离判断是否拒识为 unknown。
```

---

# 2. 研究背景与动机

现有很多细胞类型注释方法默认测试细胞都属于训练集或参考集中的已知类型，这属于 closed-set annotation。真实单细胞数据中，经常会出现未标注、未知、稀有或状态模糊的细胞类型，如果模型强行给出已知标签，就可能产生过度自信的错误注释。

scFoundation 是单细胞基础模型，论文介绍其为 100M 参数模型，覆盖约 2 万个基因，并在超过 5000 万个人类单细胞转录组 profiles 上预训练；官方仓库也提供了细胞/基因 embedding 以及 Pancreatic、PBMC 细胞类型注释任务相关代码和数据入口。([PubMed](https://pubmed.ncbi.nlm.nih.gov/38844628/?utm_source=chatgpt.com "Large-scale foundation model on single-cell transcriptomics"))

因此我们的思路是：

```text
原始单细胞表达矩阵
        ↓
scFoundation 提取 cell embedding / 细胞表征
        ↓
我们设计上层开放集判别模块
        ↓
known cell type annotation / unknown rejection
```

---

# 3. 和已有工作的区别

我们不能宣称“第一个做未知细胞识别”。已有相关方向包括：

* scNovel：聚焦 novel rare cell discovery，即新型稀有细胞发现。([OUP Academic](https://academic.oup.com/bib/article/25/3/bbae112/7636757?utm_source=chatgpt.com "scNovel: a scalable deep learning-based network for novel ..."))
* OVAAnno：利用 open-set domain adaptation 做未知细胞类型检测，主要面向 scATAC-seq，也验证了 scRNA-seq。([OUP Academic](https://academic.oup.com/bib/article/25/5/bbae370/7723313?utm_source=chatgpt.com "Detecting novel cell type in single-cell chromatin accessibility ..."))
* Conformal inference for reliable single-cell annotation：利用 conformal inference 做 OOD cell type 检测和 prediction set 不确定性量化。([OUP Academic](https://academic.oup.com/bioinformatics/article/41/10/btaf521/8257682?utm_source=chatgpt.com "Conformal inference for reliable single cell RNA-seq annotation"))

我们的差异点是：

> **不是重新提出 unknown detection，而是面向单细胞基础模型表征，构建一个轻量、可解释、可校准的开放集注释框架。**

具体优势：

```text
1. 使用 scFoundation embedding，而不是从零学习任务特定表征；
2. 使用 prototype classifier，用细胞到已知类别原型的距离判断 unknown；
3. 引入 calibration / rejection，使拒识决策更可靠；
4. 同时评估 known-cell annotation 和 unknown-cell detection；
5. 框架可插拔，后续可替换 PCA、scVI、scGPT、scFoundation 等表征。
```

---

# 4. 核心假设

目前我们的初步实验已经验证了一个很重要的现象：

```text
PCA + MLP:
closed-set accuracy 较高，但 unknown detection 很差。

PCA + Prototype:
closed-set accuracy 稍低，但 unknown AUROC 明显更好。
```

在 Zheng68K 数据上，PCA baseline 的结果为：

```text
Closed-set MLP:
accuracy = 0.6614
macro-F1 = 0.4982
balanced accuracy = 0.4806

Closed-set Prototype:
accuracy = 0.5432
macro-F1 = 0.4763
balanced accuracy = 0.5839

Open-set CD56+ NK, MLP:
known_accuracy = 0.6641
unknown_AUROC = 0.2180
unknown_AUPRC = 0.2881

Open-set CD56+ NK, Prototype:
known_accuracy = 0.5144
unknown_AUROC = 0.7221
unknown_AUPRC = 0.6220
```

这说明：

> **普通 MLP 的 softmax confidence 不适合 unknown detection；prototype distance 更适合开放集未知细胞识别。**

这就是后续论文的核心动机。

# 5. 方法框架

最终方法暂定为：

```text
CalibscAnno
```

整体流程：

```text
Input: scRNA-seq expression matrix
        ↓
Preprocessing
        ↓
scFoundation Encoder
        ↓
Cell embedding
        ↓
Projection Adapter
        ↓
Prototype Classifier
        ↓
Calibration / Rejection Module
        ↓
Output:
known cell type label / unknown label / confidence score
```

---

## 模块 1：scFoundation embedding

scFoundation 负责提供细胞表征。

这里的“表征”就是：

> 把原始几万个基因表达值转换成一个低维向量，用这个向量表示细胞的生物学状态。

形式上：

```text
Cell_i raw expression: 19264 genes
        ↓
scFoundation
        ↓
Cell_i embedding: 768-d vector
```

第一阶段只做：

```text
freeze scFoundation
extract cell embedding
```

暂时不 fine-tune 整个 scFoundation。

---

## 模块 2：Prototype classifier

每个已知细胞类型都有一个 prototype：

```text
prototype_c = 该类别所有训练细胞 embedding 的均值
```

测试细胞来了以后，计算它和所有 prototype 的相似度/距离：

```text
score(x, c) = similarity(z_x, prototype_c)
```

如果它和所有已知类别 prototype 都不够近，就可能是 unknown。

优势：

```text
1. 比普通 MLP softmax 更适合 unknown detection；
2. 可解释：能说明这个细胞离哪个已知类型最近；
3. 可视化方便：prototype UMAP、距离分布、known/unknown 分离图；
4. 和细胞类型“中心”这个生物学直觉比较一致。
```

---

## 模块 3：Calibration / Rejection

后续加入校准拒识模块。

候选方法：

```text
1. Temperature scaling
2. Threshold calibration
3. Conformal prediction score
4. Risk-coverage analysis
```

注意：conformal prediction 已经有人用于单细胞可靠注释，所以不能把它写成唯一创新。我们的写法应该是：

> **将校准拒识机制嵌入 scFoundation 表征驱动的 prototype open-set annotation 框架中。**

# 11. 论文主线表述

可以这样写：

> Existing cell type annotation methods often assume that query cells belong to known reference cell types, which may lead to overconfident misclassification when unseen cell types are present. We propose CalibscAnno, a foundation-model-based open-set annotation framework that combines scFoundation embeddings, prototype-based unknown scoring, and calibrated rejection to identify unseen cell types while maintaining accurate annotation of known cell types.

中文意思：

> 现有细胞类型注释方法通常假设查询细胞属于参考集中已知类型，当测试数据中存在未知细胞类型时，容易产生过度自信的错误注释。本文提出 CalibscAnno，一个基于单细胞基础模型表征的开放集注释框架，结合 scFoundation embedding、原型未知评分和校准拒识机制，在保证已知细胞注释准确的同时识别未知细胞类型。
